Как вычисляется рейтинг

Команда проекта «Карта полиции» по результатам общественного мониторинга, проводимого в ходе общероссийской кампании

«Гражданин и полиция», определяет параметры открытости и доступности подразделений полиции. Каждый из параметров имеет определенный вес. При подсчете рейтинга подразделения баллы по каждому параметру суммируются и переводятся в пятибалльную систему. Максимальный рейтинг подразделения — 5, он складывается из рейтинга доступности (максимум 1,5 балла) и открытости подразделения (максимум 3,5 балла). Эти две категории параметров неравновесны: всё больше услуг мы оформляем в электронном виде, и даже подать заявление о преступлении теперь можно через электронную приёмную; но впоследствии все процессуальные действия придётся совершать всё равно лично.

Если вы хотите присоединиться к команде наблюдателей и участвовать в гражданском мониторинге подразделений полиции, заполните анкету на сайте Объединенной группы общественного наблюдения (ОГОН).

Анкета, которую заполняет волонтёр во время мониторинга, доступна для скачивания по ссылке.

Подробнее о подсчете рейтинга

Доступность подразделения (1.5 балла) складывается из:

Возможность беспрепятственно попасть в подразделение (0.6 баллов)
  • Есть возможность припарковать автомобиль в 50 м. от входа;
  • Отсутствует КПП перед входом;
  • Входная дверь открывается свободно: отсутствует звонок/домофон на входе;
  • Туалеты доступны для посетителей.
Доступность для людей с ограниченными возможностями (0.9 баллов)
  • На входе в подразделение есть пандус, который по своим параметрам позволяет посетителям с ограниченными возможностями передвижения попасть внутрь;
  • Звонок находится на приемлемом для посетителей с ограниченными возможностями передвижения уровне;
  • Вход доступен для посетителей с ограниченными возможностями передвижения (отсутствует высокий порог, тяжелая дверь и другие препятствия);
  • Подразделение доступно для посетителей с ограниченными возможностями по зрению;
  • Подразделение доступно для посетителей с ограниченными возможностями по слуху;
  • Туалеты доступны для посетителей с ограниченными возможностями передвижения (отсутствуют пороги, ширина двери достаточная, есть перила).

По параметру “открытость” подразделение может получить до 3.5 баллов:

Наличие информации о работе подразделения (0.5 баллов)
  • На сайте ГУ МВД / УМВД есть имя и звание начальника подразделения;
  • На сайте ГУ МВД / УМВД есть информация о деятельности подразделения в отчетах УМВД или УУП за последние 0,5/1 год;
  • На информационном стенде есть информация о территории обслуживания подразделения и её граница.
Наличие контактов контролирующих органов и общественных правозащитных институтов (1.5 балла)
  • На информационном стенде есть контакты дежурной части районного УМВД;
  • На информационном стенде есть телефоны подразделения собственной безопасности;
  • На информационном стенде есть телефоны федеральной и региональной «горячей линии МВД»;
  • На информационном стенде есть телефоны, электронные и почтовые адреса РЕГИОНАЛЬНОГО Уполномоченного по правам ЧЕЛОВЕКА;
  • На информационном стенде есть телефоны, электронные и почтовые адреса РЕГИОНАЛЬНОГО Уполномоченного по правам РЕБЕНКА;
  • На информационном стенде есть телефоны, электронные и почтовые адреса ФЕДЕРАЛЬНОГО Уполномоченного по правам ЧЕЛОВЕКА
  • На информационном стенде есть телефоны, электронные и почтовые адреса ФЕДЕРАЛЬНОГО Уполномоченного по правам РЕБЕНКА;
  • На информационном стенде есть телефоны, электронные и почтовые адреса Общественной наблюдательной комиссии по контролю за соблюдением прав человека в местах принудительного содержания (ОНК);
  • На информационном стенде есть телефоны, электронные и почтовые адреса дежурного прокурора районной и региональной прокуратуры;
  • На информационном стенде есть телефоны, электронные и почтовые адреса регионального Общественного совета при ГУ МВД и УМВД;
  • На информационном стенде есть информация о присутствии членов Общественных советов при приёме граждан (график встреч).
Открытость: наличие памяток о действиях в случае нарушений со стороны сотрудников полиции (1.5 балла)
  • На информационном стенде есть информация о порядке подачи заявлений (обязательность выдачи талона КУСП, сроки рассмотрения заявления, выдержки из нормативных актов);
  • На информационном стенде есть информация о том, что делать посетителю, если не приняли заявление;
  • На информационном стенде есть информация о том, как обжаловать действия сотрудников полиции;
  • На информационном стенде есть информация о правах задержанных.
