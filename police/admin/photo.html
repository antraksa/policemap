Загрузка фото

Желательно, чтобы имя файла не содержало пробелы и отражало его суть. Назовите файл по аналогии с подразделениями, где уже есть фото. Например, для СПб используются имена типа OP_01_ph1_gsw.jpg. Это фото №1 для ОП №1. По аналогии фото №1 для ОП №75 нужно назвать OP_75_ph1_gsw.jpg. По ссылкам можно посмотреть список загруженных файлов для Санкт-Петербурга или Воронежа.

Выберите город, для которого загружаете.

Важно

  • Максимальный размер файла - 300 KB.
  • Разрешены только изображения типа JPG, GIF, PNG.
  • Не больше 1500 пикселей по длинной стороне.
  • Смысловой центр фото должен находиться в центре.
  • Желательно квадратные, правильно повернутые
